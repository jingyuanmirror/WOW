## 1. 页面概述

### 1.1 页面定位
首页是用户接触品牌的第一窗口，承担着品牌形象展示、产品价值传递、用户转化引导的核心职责。通过精心设计的多个功能区块，让访客在3-5秒内理解品牌价值，并产生进一步探索的兴趣。

### 1.2 页面目标
- **品牌认知**: 建立专业、高质量的品牌形象
- **用户转化**: 引导访客进入作品集页面或直接购买
- **信息传递**: 快速传达核心价值主张和产品优势
- **用户留存**: 通过Newsletter订阅、社区引导建立长期联系
- **SEO优化**: 作为主要落地页，承担搜索引擎流量入口功能

### 1.3 核心指标 (KPI)
- **首屏停留时间** > 5秒
- **跳出率** < 40%
- **页面向下滚动率** > 60%
- **CTA点击率** > 15%
- **首页到作品集转化率** > 30%
- **注册/订阅转化率** > 5%
- **页面加载时间** < 2秒

## 2. 页面结构与功能模块

### 2.1 导航栏 (Navigation Bar)

#### 2.1.1 功能描述
全局导航栏，固定在页面顶部，提供快速访问各主要页面的入口。

#### 2.1.2 布局结构
```
[Logo]  [作品集] [使用文档] [关于] [社区]  [搜索] [购物车(0)] [登录/头像]
```

#### 2.1.3 详细功能

**左侧区域**:
- **品牌Logo**: 
  - 点击返回首页
  - 支持SVG格式，适配深色/浅色主题
  - 尺寸: 120px x 40px
  - 包含品牌标识与名称

**中央导航菜单**:
- **作品集** (`/skins`)
  - 下拉菜单显示皮肤分类:
    - 全部皮肤
    - PvP专用
    - PvE副本
    - 治疗界面
    - DPS输出
    - 免费皮肤
- **使用文档** (`/docs`)
  - 下拉菜单:
    - 安装指南
    - 自定义教程
    - 常见问题
    - 视频教程
- **关于** (`/about`)
  - 关于设计师
  - 设计理念
  - 联系方式
- **社区** (`/community`)
  - 用户作品展示
  - 讨论区
  - Discord服务器

**右侧功能区**:
- **搜索图标**: 
  - 点击展开搜索框
  - 支持皮肤名称、标签搜索
  - 实时搜索建议
- **购物车图标**: 
  - 显示商品数量徽章
  - 悬停显示迷你购物车预览
  - 点击跳转到购物车页面
- **用户区域**:
  - 未登录: "登录" / "注册" 按钮
  - 已登录: 用户头像 + 下拉菜单
    - 我的账户
    - 订单历史
    - 已购买皮肤
    - 下载中心
    - 设置
    - 退出登录

#### 2.1.4 交互特性
- **滚动行为**: 向下滚动时收缩，向上滚动时展开
- **背景效果**: 滚动到内容区时添加半透明背景 + 模糊效果
- **移动端**: 汉堡菜单，侧边滑出导航
- **活跃状态**: 当前页面导航项高亮显示

#### 2.1.5 技术实现
```typescript
// 组件结构
<Navigation>
  <Logo />
  <NavMenu items={navItems} />
  <SearchBar />
  <CartIcon count={cartCount} />
  <UserMenu user={currentUser} />
</Navigation>
```

### 2.2 英雄区 (Hero Section)

#### 2.2.1 功能描述
首屏核心视觉区域，通过震撼的视觉效果和简洁有力的文案，在3秒内抓住用户注意力，传递品牌核心价值。

#### 2.2.2 布局结构
```
[全屏背景动画/视频]
    [主标题]
    [副标题/价值主张]
    [主CTA按钮] [次CTA按钮]
    [信任指标 - 用户数/下载数/评分]
    [向下滚动提示动画]
```

#### 2.2.3 视觉设计

**背景层**:
- **选项A**: 循环播放的皮肤演示视频 (WebM格式，自动播放，静音)
- **选项B**: 粒子动画效果 (Three.js / Canvas)
- **选项C**: 精选皮肤截图轮播 (Carousel)
- **推荐**: 选项A + 渐变遮罩层，确保文字可读性
- **尺寸**: 全屏高度 (100vh)
- **色调**: 深色基调 (#0a0a0a) + 金色点缀

**内容层**:
- **主标题**: 
  - 文案示例: "重新定义你的战斗体验"
  - 字体: 56px - 72px, 粗体, 渐变色 (金→橙)
  - 动画: 淡入 + 轻微上移效果
  - 延迟: 0.2s
  
- **副标题**:
  - 文案示例: "专业设计 | 极致美观 | 简单易用 | 10万+玩家的选择"
  - 字体: 20px - 24px, 常规, 白色 80%透明度
  - 动画: 淡入效果
  - 延迟: 0.4s

#### 2.2.4 CTA按钮组

**主按钮 (Primary CTA)**:
- **文案**: "探索皮肤作品" 或 "查看作品集"
- **样式**: 
  - 大尺寸按钮 (px-8 py-4)
  - 金色渐变背景
  - 白色文字, 粗体
  - 悬停效果: 放大1.05倍 + 发光效果
  - 图标: 右箭头 →
- **跳转**: `/skins`
- **动画**: 淡入 + 上移, 延迟0.6s

**次按钮 (Secondary CTA)**:
- **文案**: "观看演示视频" 或 "了解更多"
- **样式**:
  - 大尺寸按钮
  - 透明背景 + 白色边框
  - 白色文字
  - 悬停效果: 白色背景 + 深色文字
  - 图标: 播放按钮 ▶
- **行为**: 打开视频模态框
- **动画**: 淡入 + 上移, 延迟0.8s

#### 2.2.5 信任指标 (Social Proof)

显示位置: CTA按钮下方

**指标展示**:
```
[图标] 100,000+ 活跃用户  |  [图标] 50+ 精品皮肤  |  [图标] 4.9/5.0 用户评分
```

- **数据源**: 动态从数据库获取
- **动画**: 数字递增动画 (CountUp.js)
- **样式**: 白色文字, 半透明背景, 圆角边框
- **延迟**: 1.0s

#### 2.2.6 向下滚动提示

- **位置**: Hero区底部中央
- **样式**: 向下箭头图标 + "探索更多" 文字
- **动画**: 上下浮动动画 (无限循环)
- **行为**: 点击平滑滚动到下一区块

#### 2.2.7 技术实现要点

```typescript
// Hero组件结构
<HeroSection>
  <BackgroundVideo src="/videos/hero-demo.webm" />
  <ContentOverlay>
    <h1 className="hero-title">重新定义你的战斗体验</h1>
    <p className="hero-subtitle">专业设计 | 极致美观 | 简单易用</p>
    <CTAButtons>
      <PrimaryButton href="/skins">探索皮肤作品</PrimaryButton>
      <SecondaryButton onClick={openVideoModal}>观看演示</SecondaryButton>
    </CTAButtons>
    <SocialProof stats={statsData} />
  </ContentOverlay>
  <ScrollIndicator />
</HeroSection>
```

**性能优化**:
- 背景视频压缩优化 (< 5MB)
- 使用 `loading="lazy"` 延迟加载非首屏资源
- 预加载关键资源 (字体、图标)


**详细元素**:

1. **预览媒体**:
   - 16:9 宽高比
   - 高清图片 (1920x1080, WebP格式)
   - 悬停效果: 播放GIF/短视频展示交互效果
   - 渐变遮罩 (底部渐黑)
   - 圆角: 12px

2. **品质标签**:
   - 位置: 左上角
   - 颜色对应魔兽品质系统:
     - 优秀 (绿色): #1EFF00
     - 精良 (蓝色): #0070DD
     - 史诗 (紫色): #A335EE
     - 传说 (橙色): #FF8000
   - 样式: 小标签, 半透明背景

3. **皮肤名称**:
   - 字体: 24px, 粗体
   - 颜色: 白色
   - 限制: 单行显示, 超出省略

4. **简短描述**:
   - 字体: 14px, 常规

   - 限制: 最多2行, 超出省略

6. **价格区域**:
   - 字体: 28px, 粗体
   - 颜色: 金色 (#FFD700)
   - 免费皮肤显示: "免费" 标签
   - 折扣显示: 
     - 原价划线
     - 折扣价突出
     - 折扣标签 (-20%)

7. **按钮组**:
   - **查看详情**: 次要按钮, 透明背景
   - **立即购买**: 主要按钮, 金色背景

## 4. 性能优化

### 4.1 加载优化

**关键指标**:
- **FCP** (First Contentful Paint) < 1.5s
- **LCP** (Largest Contentful Paint) < 2.5s
- **TTI** (Time to Interactive) < 3.5s
- **CLS** (Cumulative Layout Shift) < 0.1

**优化策略**:

1. **图片优化**:
   - 使用 Next.js Image 组件
   - WebP/AVIF 格式
   - 响应式图片 (srcset)
   - 懒加载 (loading="lazy")
   - 优先加载首屏图片

2. **代码分割**:
   - 动态导入非首屏组件
   - 路由级代码分割
   - 组件懒加载

3. **字体优化**:
   - 字体子集化
   - font-display: swap
   - 预加载关键字体

4. **JavaScript优化**:
   - 最小化bundle体积
   - Tree shaking
   - 去除未使用代码

5. **CSS优化**:
   - 关键CSS内联
   - 非关键CSS异步加载
   - 使用Tailwind的purge功能

### 4.2 缓存策略

```javascript
// Next.js配置
export default {
  images: {
    formats: ['image/avif', 'image/webp'],
    deviceSizes: [640, 750, 828, 1080, 1200, 1920],
  },
  
  headers: async () => [
    {
      source: '/images/:path*',
      headers: [
        {
          key: 'Cache-Control',
          value: 'public, max-age=31536000, immutable',
        },
      ],
    },
  ],
}
```
