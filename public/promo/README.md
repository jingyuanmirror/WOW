# 活动宣传图片目录

将活动宣传的大图放在此目录下。

## 推荐规格

- **尺寸**: 1920×600px 或更大
- **格式**: JPG, PNG, WebP
- **文件大小**: 建议 < 500KB
- **命名**: 使用有意义的名称，如 `new-year-2025.jpg`, `summer-sale.png` 等

## 使用方法

1. 将图片文件放到这个目录
2. 在 `src/lib/data.ts` 的 `promoSlides` 中引用：

```typescript
{
  id: 1,
  title: "我的活动",
  image: "/promo/my-banner.jpg",  // 使用 /promo/ 开头
  // ...
}
```

## 当前使用的图片

目前配置使用的是 Unsplash 的外部链接作为示例。您可以：

1. **替换为本地图片**: 下载或创建自己的图片，放到此目录，然后更新 `data.ts` 中的路径
2. **继续使用外部链接**: 如果使用外部图片托管服务，需要在 `next.config.ts` 中配置允许的域名

## Next.js 图片优化

Next.js 会自动优化此目录下的图片，提供：
- 自动格式转换（WebP）
- 按需加载
- 尺寸优化
- 响应式图片

无需手动处理！
